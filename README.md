# 🎬 Cinéma Archive

**Cinéma Archive** is a premium web application designed for film enthusiasts, TV series binge-watchers, and anime fans. It provides a sophisticated interface to search for titles, explore detailed metadata, and find where to watch your favorite content.

![Cinéma Archive Hero](https://images.unsplash.com/photo-1489599849927-2ee91cede3ba?q=80&w=2070&auto=format&fit=crop)

## ✨ Features

- **Premium UI/UX**: A cinematic aesthetic with glassmorphism, micro-animations, and a responsive design system.
- **Unified Search**: Search across Movies, TV Series, and Anime using a single, intuitive interface.
- **Deep Metadata**: Detailed information including plots, ratings (IMDb, Rotten Tomatoes), cast, episodes, and reviews.
- **Where to Watch**: Quick links to popular streaming platforms (Netflix, Crunchyroll, Disney+, etc.).
- **Live Trending**: Real-time trending anime list powered by the Jikan API.

## 🚀 Technologies

- **HTML5**: Semantic structure and modern SEO practices.
- **Vanilla CSS**: Custom design system with advanced animations and HSL-based coloring.
- **ES6+ JavaScript**: Modern, modular application logic with async/await and robust API integration.
- **APIs**: Powered by [OMDB](https://www.omdbapi.com/) and [Jikan (MAL)](https://jikan.moe/).

## 🛠️ Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/cinema-archive.git
   ```
2. **Open the project**:
   Simply open `index.html` in any modern web browser.
3. **Configuration**:
   The project uses a default OMDB API key. For production use, please consider obtaining your own key at [omdbapi.com](https://www.omdbapi.com/apikey.aspx).
